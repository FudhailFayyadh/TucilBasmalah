# Queens Solver

A command-line puzzle solver for N-Queens problem with color constraints using brute force algorithm.

## Requirements

- Python 3.x (or compatible version)

## Installation & Compilation

1. Clone or download this repository.
2. From the repository's root directory, navigate to src folder:
   ```
   cd src
   ```

## Usage

### Command Line

1. Open a terminal in the repository root.
2. Run
   ```
   cd src
   python main.py
   ```
